# Posture Reminder - 자세 교정 리마인더

> 올바른 자세 유지를 위한 macOS 데스크톱 애플리케이션

## 📝 프로젝트 소개

장시간 컴퓨터 작업 시 나쁜 자세로 인한 건강 문제를 예방하기 위한 자세 교정 리마인더 앱입니다.
카메라를 통해 사용자의 자세를 실시간으로 인식하고, 기준 자세와 비교하여 자세 교정을 유도합니다.

## ✨ 주요 기능

### 1. AI 기반 자세 인식
- **Vision Framework** 활용한 실시간 포즈 감지
- 19개 신체 관절 포인트 추적
- 정확한 자세 비교 및 유사도 측정

### 2. 직관적인 오버레이 UI
- 기준 자세 이미지를 배경으로 표시
- 실시간 카메라 화면을 투명도 조절 가능한 오버레이로 제공
- 두 화면을 겹쳐 보면서 정확한 자세 매칭 가능
- 스켈레톤 시각화로 관절 위치 확인

### 3. 스마트 리마인더
- 사용자 설정 간격으로 자세 확인 알림
- 리마인더 완료 후부터 타이머 재시작 (자연스러운 워크플로우)
- 특정 앱 실행 중 리마인더 차단 기능 (예: Zoom 회의 중)

### 4. 사용자 친화적 인터페이스
- 실시간 투명도 조절 슬라이더
- 자세 유사도 퍼센트 표시
- 색상 코딩 피드백 (빨강/노랑/초록)

## 🚀 설치 방법

1. `PostureReminder.dmg` 파일을 다운로드
2. DMG 파일을 열고 `stop_the_world.app`을 Applications 폴더로 드래그
3. 앱 실행 시 카메라 권한 허용

## 💡 사용 방법

### 초기 설정
1. 앱 실행 후 **"자세 설정"** 버튼 클릭
2. 올바른 자세를 취하고 카메라에 상체가 보이도록 조정
3. 19개 관절이 감지되면 **"자세 기록하기"** 버튼 클릭

### 리마인더 사용
1. 메인 화면에서 **"리마인더 시작"** 버튼 클릭
2. 설정된 간격(기본 20초)마다 리마인더 창 표시
3. 리마인더 창에서:
   - 기준 자세 이미지가 배경에 표시
   - 실시간 카메라 화면이 투명하게 오버레이
   - 우측 하단 슬라이더로 카메라 투명도 조절 (30%-100%)
   - 두 화면을 맞춰가며 자세 교정
4. 유사도 50% 이상 달성 시 자동으로 창 닫힘
5. 창이 닫힌 시점부터 다시 타이머 시작

## 🛠 기술 스택

- **Framework**: Flutter (Dart)
- **ML/AI**: Apple Vision Framework
- **Platform**: macOS 11.0+
- **카메라**: AVFoundation

## 📋 시스템 요구사항

- macOS 11.0 (Big Sur) 이상
- 내장 또는 외장 카메라
- 카메라 접근 권한

## 🎯 특징

### 혁신적인 오버레이 방식
기존의 작은 미리보기 이미지 방식과 달리, 기준 자세와 현재 자세를 전체 화면에서 직접 겹쳐볼 수 있어 훨씬 정확한 자세 교정이 가능합니다.

### 스마트한 타이머 로직
단순히 고정된 시간 간격이 아닌, 사용자가 자세를 교정하고 리마인더를 완료한 시점부터 타이머가 시작되어 더 자연스러운 사용 경험을 제공합니다.

### 실시간 피드백
스켈레톤 오버레이와 유사도 퍼센트, 색상 코딩을 통해 현재 자세 상태를 한눈에 파악할 수 있습니다.
